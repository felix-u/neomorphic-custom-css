### Neomorphism is a modern design language that uses shadow and light to give the illusion of depth.

These themes can be applied using the **Stylus** extension for Firefox and Chromium-based browsers. Colours are easily modified by changing the root variables at the top of each CSS file. Example screenshots use ![gruvbox-material](https://github.com/sainnhe/gruvbox-material).
